ColorMode

Enum

Namespace: ANSITerm

Declared in: ANSITerm

Defines a color mode. Integer value of this enum corresponds to the number of colors it defines (e.g. Color8 is equal to 8).

Enumeration

Underlying type: int

Values:

  • Color256 = 256
